# 디자인 레퍼런스 (UI 전면 개편용)

사장님이 원하는 화면 디자인을 **이미지로 모으는 곳**입니다.
여기에 올라온 그림을 보면서 Claude가 UI를 개편합니다.

## 넣는 방법
1. 아래 화면별 폴더 중 맞는 곳을 고른다.
2. 그 폴더로 이미지를 **드래그&드롭** (여러 장 OK).
3. 설명할 게 있으면 이 README 맨 아래 "메모"에 한 줄 적거나, 폴더 안에 `메모.txt`로 적어도 됨.

> 파일 이름은 아무거나 OK (예: `1.png`, `로그인-위쪽.jpg`).
> 화면이 헷갈리면 `99-etc/`에 넣고 메모에 "이건 ○○화면" 적어주세요.
