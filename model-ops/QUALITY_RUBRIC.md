# QUALITY_RUBRIC.md — 결과물 평가 기준 (100점)
> 사용법: 결과물 제출 전 반드시 자체 채점하고, 보고에 총점과 항목별 점수를 붙인다.
> **80점 미만 = 배포 금지(재작업). 90점 미만 = 메인 화면(홈/랜딩 첫 화면) 사용 금지.**
> 채점은 후하게 주지 않는다. 애매하면 낮은 점수를 준다. 감점 사유를 반드시 한 줄씩 적는다.

## 배점표 (12항목, 총 100점)

### 1. 모바일 완성도 — 12점
- 375px에서 잘림·겹침·가로스크롤 없음 (4)
- 폴드 위에 정체성+이미지/숫자+CTA 모두 노출 (4)
- 터치 타깃 ≥44px, 고정 바가 콘텐츠를 가리지 않음 (4)
- 즉시 0점 처리: 375px에서 가로스크롤 발생 시 이 항목 0점.

### 2. PC 완성도 — 8점
- 1280px에서 콘텐츠 최대폭 제한 + 중앙 정렬(늘어짐 없음) (4)
- 모바일→PC 전환 시 그리드·이미지 비율 유지, 어색한 빈 공간 없음 (4)

### 3. CTA 강도 — 10점
- 1차 CTA가 페이지당 1종류, 폴드 위 노출 (4)
- 라벨이 행동+대상("매입 신청" 등), 모호 라벨 없음 (3)
- 페이지 끝 CTA 재노출 + 대체 연락 수단(전화/카카오) (3)

### 4. 전환율 구조 — 10점
- 방문자 흐름이 설계됨: 첫 화면 → 신뢰 근거 → 절차 → CTA (4)
- 모든 섹션에 목적(전환/신뢰/정보/탐색)이 있고 불필요 섹션 없음 (3)
- 전환 직전 마찰 제거: 폼 필수 입력 최소(≤5), 전화·카카오 원터치 (3)

### 5. 정보 위계 — 8점
- 3초 안에 "무엇을 하는 곳"인지 읽힘 (3)
- 폰트 크기 5단계 이내, 크기가 중요도와 일치(핵심 숫자·가격이 가장 큼) (3)
- 한 섹션 본문 ≤3문장, 긴 설명은 접힘 처리 (2)

### 6. 이미지 활용 — 8점
- 히어로 실물 이미지 폭 100%, 선명·비율 유지 (3)
- AI 티 나는 아이콘/이모지 아이콘/스톡 악수 사진 없음 (3)
- 이미지 라운드·비율 통일, alt·lazy 처리 (2)

### 7. 카피 명확성 — 8점
- 헤드라인 ≤15자, 서비스 정체성 즉시 전달 (3)
- 과장·보장 문구 없음, 근거 있는 사실·숫자 사용 (3)
- 기업형 전문 톤(유치한 표현 없음), 오탈자 없음 (2)

### 8. 브랜드 신뢰감 — 8점
- 절차 스텝(3~5단계) 시각화 존재 (3)
- 사업자 정보·연락처·주소 노출 (2)
- 업종에 맞는 톤(고급 커머스/정비센터/매입 서비스)이 첫 화면부터 느껴짐 — 템플릿 느낌이면 0점 (3)

### 9. 디자인 규격 일관성 — 8점
- 같은 요소(버튼·카드·입력칸) 높이·패딩·라운드·폰트 픽셀 단위 동일 (4)
- 색 6개 이내, 무근거 그라데이션 없음, 좌측 시작선 3종 이내 (4)

### 10. 코드 안정성 — 8점
- 콘솔 에러 0, 깨진 링크·이미지 0 (3)
- flex 잘림(min-width), z-index 겹침 등 알려진 함정 처리 (3)
- 폼 검증·에러 상태 존재(실패 시 사용자에게 보이는 메시지) (2)

### 11. 유지보수성 — 6점
- 기존 클래스·토큰 재사용, 일회용 인라인 스타일 없음 (3)
- 중복 코드 없음(같은 값이 두 곳에 하드코딩되어 있지 않음) (3)

### 12. 디테일 완성도 — 6점
- 줄바꿈·말줄임 처리(line-clamp), 긴 텍스트에도 안 깨짐 (2)
- 빈 상태·로딩 상태 디자인 존재 (2)
- 간격·정렬 미세 어긋남 없음(1~2px 어긋남도 감점) (2)

## 등급 판정
| 총점 | 판정 |
|---|---|
| 90~100 | 메인 화면 포함 배포 가능 |
| 80~89 | 서브 페이지 배포 가능. 메인 화면은 재작업 |
| 70~79 | 배포 금지. 감점 상위 3개 항목부터 재작업 |
| ~69 | 구조 재설계. 부분 수정으로 살리지 말 것 |
