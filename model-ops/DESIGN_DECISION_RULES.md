# DESIGN_DECISION_RULES.md — 디자인 판단 규칙
> 목적: "깔끔하게, 고급스럽게" 같은 감각 판단을 수치·조건 판단으로 바꾼다.
> 모든 규칙은 측정 가능해야 한다. 측정할 수 없는 규칙은 이 문서에 넣지 않는다.
> 기준 뷰포트: 모바일 375×812, 태블릿 768, PC 1280.

## 0. 최상위 판단 순서
디자인 결정이 충돌하면 이 순서로 이긴다:
**① 전환(문의·예약·구매가 일어나는가) → ② 신뢰(전문 업체로 보이는가) → ③ 정보 위계(3초 안에 읽히는가) → ④ 미감.**
"예쁜데 CTA가 약한 안"과 "평범한데 CTA가 명확한 안" 중에서는 후자를 고른다.

## 1. 모바일 첫 화면 (폴드 위, 375×812 기준)
폴드 위에 반드시 들어갈 것 — **정확히 이 4개, 그 이상 넣지 않는다:**
1. 로고/브랜드명 (헤더)
2. 헤드라인 1개: 무엇을 하는 곳인지. 공백 포함 한글 ≤15자. (예: "명품시계, 오늘 최고 조건으로 매입")
3. 뒷받침 요소 1개: 실물 이미지(폭 100%) **또는** 핵심 숫자 1개(예: "평균 감정 30분"). 둘 다 넣으려면 숫자를 이미지 위에 오버레이.
4. 1차 CTA 버튼 1개.
- 폴드 위 총 텍스트 ≤4줄(헤더 제외). 5줄 이상이면 실패.
- 판정법: 375×812로 열고 스크롤 0 상태 스크린샷 → 위 4개가 다 보이고 다른 건 없으면 통과.

## 2. 레이아웃
- 모바일 좌우 패딩: 20px 고정. 섹션마다 다르게 두지 않는다.
- 섹션 간 세로 간격: 48~64px 중 하나로 통일. 섹션 내부 요소 간격: 10~12px.
- PC 콘텐츠 최대폭: 페이지 1200px, 리스트 720px, 폼 480~560px. 초과 시 중앙 정렬 + 좌우 여백. 풀블리드는 히어로 이미지·배경색 밴드만 허용.
- 그리드: 카드 나열은 모바일 1~2열, PC 3~4열. 한 행에 카드가 5개 이상이면 실패.
- 판정법: 어떤 요소든 좌측 시작선이 3종류를 넘으면(패딩선/카드 내부선/들여쓰기선) 정렬 실패.

## 3. 타이포그래피
- 폰트: Pretendard 1종. 장식용 세리프는 로고 워드마크에만 허용.
- 크기 단계는 **정확히 5단계만** 사용: 28/20/16/14/12px (모바일 기준. PC 히어로만 40~48px 허용).
  이 5단계 밖의 크기(17px, 15px, 13px…)가 코드에 있으면 실패.
- 굵기: 700(제목·CTA·숫자 강조) / 400(본문) 2단계. 300, 500, 600 혼용 금지.
- 본문 행간 1.5~1.6, 제목 행간 1.2~1.3. 본문 한 줄 ≤ 한글 35자.
- 숫자: `font-variant-numeric: tabular-nums`, 본문과 동일 크기 이상. 가격·건수 등 핵심 숫자는 주변 텍스트보다 1.5배 이상 크게.
- 텍스트량 상한: 한 섹션 본문 ≤3문장. 넘으면 ①리스트화 ②접기(더보기) ③삭제 순으로 처리.

## 4. 색
- 팔레트는 **최대 6개**: 배경 1(#fff 또는 #f4f1ea 계열) + 텍스트 2(#1a1a1a, #6b6b6b) + 보더 1(#e5e3df) + 브랜드/CTA 1 + 상태(빨강 배지 등) 1. 7번째 색이 필요하면 기존 색을 지워라.
- CTA 색은 전 페이지에서 1개로 고정(기본 검정 #111). CTA 색을 CTA 아닌 곳에 쓰지 않는다.
- 그라데이션: 의미(사진 위 텍스트 가독용 오버레이) 없는 그라데이션 금지. 보라→파랑류 배경 그라데이션은 무조건 금지.
- 레퍼런스 이미지가 주어진 화면은 레퍼런스의 색을 그대로 쓴다. 임의 흑백화·톤 변경 금지.

## 5. 버튼
- 1차 CTA: 모바일 풀폭(패딩 안쪽), 높이 52~56px, 라운드 12px, 16px 700, 브랜드색 배경+흰 글씨. 페이지당 1종류.
- 2차: 같은 높이·라운드, 흰 배경+#e5e3df 보더+검정 글씨. 3차(텍스트 링크)까지만 허용 — 버튼 위계 4단계 이상 금지.
- 비활성: 배경 #eceae6, 글씨 #9a9a9a. 투명도(opacity)로 비활성 표현 금지.
- 나란히 배치 시 높이·라운드 동일, 간격 10px. 터치 타깃 최소 44×44px.
- 라벨은 행동+대상 2~6자: "매입 신청" "정비 예약" "시세 조회". 금지 라벨: "더 알아보기" "클릭" "GO" "확인하기".
- 하단 고정 CTA 바를 쓰면 본문 마지막에 그 높이만큼 padding-bottom을 준다(콘텐츠 가림 금지).

## 6. 이미지
- 히어로: 실물 사진(제품·매장·작업 현장), 모바일 폭 100%, 높이는 뷰포트의 40~60%. 스톡 악수 사진·외국인 모델·추상 3D 금지.
- AI 생성 티가 나는 아이콘/배너/일러스트 금지. 이모지를 UI 아이콘으로 쓰지 않는다. 아이콘이 필요하면 단색 라인 아이콘 1세트(같은 스트로크 굵기)로 통일.
- 사진 위 텍스트는 하단 40% 어두운 오버레이(rgba(0,0,0,.35~.5)) 위에만 올린다. 오버레이 없이 사진 위 흰 글씨 금지.
- 모든 이미지: 라운드 값 통일(카드 내 이미지 = 카드 라운드와 일치), `object-fit: cover`, 고정 비율(잘림/찌그러짐 금지), `alt` 필수, 폴드 아래는 `loading="lazy"`.
- 이미지:텍스트 면적비 — 상품·사례 중심 페이지는 화면의 50% 이상이 이미지여야 한다. 스크롤해도 사진이 안 나오는 구간이 2화면 이상 이어지면 실패.

## 7. 입력 필드 (폼)
- 구조: 라벨(위, 12~14px 회색) → 입력 박스. 플레이스홀더를 라벨 대용으로 쓰지 않는다.
- 박스: 높이 52px, 좌우 패딩 16px, 라운드 12px, 보더 1px #e5e3df, 글자 16px(iOS 확대 방지 — 16px 미만 금지). 포커스 보더 #1a1a1a.
- 인라인 액션(지우기 ✕, 비밀번호 👁, 인증 버튼)은 박스 안 오른쪽 끝. 같은 줄 보조 버튼은 높이 52px 동일.
- 단계형(토스식) 폼: 상단 '‹ 뒤로'+제목 → 큰 질문 제목(24~26px 700) → 입력 1~N → 하단 고정 풀폭 CTA.
- 한 화면에 입력칸 5개 초과 시 단계를 나눈다.
- 금액 입력칸: 입력 보조로만 오른쪽에 `= 140만원` 한글 읽기 표시. **표시(디스플레이) 화면에는 한글 병기 금지.**

## 8. 카드
- 흰 배경, 라운드 14px, 보더 1px #e5e3df. 그림자는 없음 또는 `0 1px 3px rgba(0,0,0,.06)` 이하.
- 같은 목록의 카드는 높이·이미지 비율·텍스트 줄수 완전 동일. 텍스트 길이 차이는 `line-clamp`로 자른다.
- 카드 내부: 이미지 → 제목(1~2줄) → 핵심 정보 1줄(가격/연식 등) → (선택) 상태 뱃지. 4요소 초과 금지.
- 상품 카드의 가격은 카드에서 가장 굵고 큰 텍스트여야 한다.

## 9. 섹션 구성
- 모든 섹션은 시작 전에 목적을 하나 적을 수 있어야 한다: **전환 / 신뢰 / 정보 / 탐색** 중 1개. 답 못 하면 그 섹션은 만들지 않는다.
- 같은 레이아웃 패턴(예: 아이콘+제목+설명 3열)을 한 페이지에 2번 이상 반복 금지.
- 섹션 제목: 16~20px 700, 좌측 정렬 기본. 제목 아래 부연은 1줄까지.
- 페이지 섹션 수: 랜딩 5~7개, 그 이상이면 합치거나 하위 페이지로 뺀다.
- 페이지 끝에는 반드시 전환 섹션(CTA 재노출 + 전화/카카오 등 대체 연락 수단)을 둔다.

## 10. 신뢰 요소 (모든 사업 공통)
- 절차는 번호 스텝(3~5단계)으로 시각화: "신청 → 방문/입고 → 견적 → 정산/완료". 각 스텝 설명 ≤1줄.
- 사업자 정보·주소·연락처를 푸터에 상시 노출. 실적 숫자는 검증 가능한 것만(출처·기간 명시 가능해야 함).
- 후기·사례는 실제 사진+구체 정보(모델명, 작업 내용) 포함일 때만 싣는다. 지어낸 후기 금지.
- 금지 문구: "업계 최고가 보장" "100% 만족" "무조건" "최저가 보장". 대신 근거 있는 사실 서술: "구성품·상태 확인 후 10분 내 확정 견적".

## 11. 좋은 디자인 / 나쁜 디자인 판별 요약표
| 항목 | 좋음 (통과) | 나쁨 (재작업) |
|---|---|---|
| 첫 화면 | 정체성+이미지/숫자+CTA, 텍스트 ≤4줄 | 인사말·긴 소개문, CTA 폴드 아래 |
| 폰트 크기 | 5단계 이내 | 6종류 이상 뒤섞임 |
| 색 | 6색 이내, CTA색 1개 | 무근거 그라데이션, 색 7개+ |
| 버튼 | 행동+대상 라벨, 규격 통일 | "더 알아보기", 높이 제각각 |
| 이미지 | 실물 사진 크게 | 작은 스톡/AI 아이콘 나열 |
| 섹션 | 각자 목적 1개, 패턴 반복 없음 | 3열 아이콘 패턴 반복 |
| 카피 | 구체 사실·숫자 | 과장 보장·추상 형용사 |
| 정렬 | 시작선 3종 이내, 규격 픽셀 일치 | 들쭉날쭉, 줄바꿈 어긋남 |
