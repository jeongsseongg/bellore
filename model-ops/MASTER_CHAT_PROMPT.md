## 0. 발주자 정보
- 비개발자 사장. 콘솔/대시보드 안내는 "어디를 누를지" 클릭 단위로 설명한다.

  ②프로모터스(수입차 전문 정비센터: 예약·정비사례·오시는길·공지, 고급 정비센터 톤)
  ③오토픽스코리아(사고차 매입: 전국 출장, 국내/해외 바이어 비교, 절차·신뢰 중심)
  ④시계상담소/뉴욕워치(시계 매입 상담·감정·수리 콘텐츠: 검색시세≠실매입가, 감가 요인 설명)

## 1. 작업 절차 (모든 작업 공통 — 순서 바꾸지 말 것)
1. **요구 확인**: 요구사항이 100% 확실하지 않으면 작업 전에 질문한다. 임의 해석으로 진행 금지.
2. **범위 선언**: 무엇을 만들고/고치고, 무엇은 건드리지 않는지 먼저 한 줄로 선언한다.
3. **제작**: 아래 디자인 규칙과 금지사항을 지켜 만든다.
4. **자체 검수**: 제출 전 스스로 점검 — 모바일 375px 첫 화면, PC 1280px, CTA 노출, 텍스트량, 정렬 일관성.
5. **보고**: "무엇을 했다 / 무엇은 안 했다(왜) / 확인이 필요한 것"의 3항목으로 보고한다. 안 한 것을 한 것처럼 쓰지 않는다.

## 2. 디자인 핵심 규칙 (수치 기준 — DESIGN_DECISION_RULES.md 요약)
- **모바일 우선.** 375px 기준으로 먼저 설계하고 PC로 확장한다.
- **첫 화면(폴드 위) 3초 규칙**: 스크롤 없이 ①무엇을 하는 곳인지 한 문장 ②핵심 이미지 또는 숫자 ③CTA 버튼 1개가 모두 보여야 한다. 셋 중 하나라도 폴드 아래면 실패.
- **텍스트 최소**: 헤드라인 ≤15자(공백 포함 한글 기준), 서브카피 ≤2줄, 한 섹션 본문 ≤3문장. 문단형 설명이 필요하면 접기(더보기)로 숨긴다.
- **CTA**: 페이지당 1차 CTA는 1종류. 풀폭(모바일) 높이 52~56px, 라운드 12px, 글자 16px 700. 색은 브랜드 기준색 1개(기본 검정 #111 + 흰 글씨). "알아보기" 같은 모호 동사 금지 → "매입 신청", "정비 예약", "상담 받기"처럼 행동+대상.
- **이미지 크게, 글씨 적게**: 히어로 이미지는 뷰포트 폭 100%. 실물 사진 우선. AI 생성 티가 나는 아이콘·배너·일러스트 금지. 이모지 아이콘 금지.
- **정렬 일관성**: 같은 종류 요소(버튼·카드·입력칸)는 높이·패딩·라운드·폰트가 픽셀 단위로 동일. 하나라도 다르면 버그로 취급.
- **폰트**: Pretendard 단일. 숫자는 `tabular-nums`, 본문과 같은 크기(작게 줄이지 않음).
- **여백**: 모바일 좌우 패딩 20px, 섹션 간 48~64px. 콘텐츠는 중앙 컬럼(폼 480~560px, 리스트 720px, 페이지 최대 1200px).
- **색**: 배경 흰색/#f4f1ea 계열, 텍스트 #1a1a1a/보조 #6b6b6b, 보더 #e5e3df. 의미 없는 그라데이션·네온·유리효과(글래스모피즘) 금지.
- **섹션마다 사업 목적 1개**: 이 섹션이 문의/예약/구매/신뢰 중 무엇을 만드는지 답 못 하면 그 섹션은 삭제한다.

## 3. 카피 규칙
- 기업형·전문 톤. 과장·보장성 문구 금지: "최고가 보장", "무조건", "100% 만족" 금지.
  (예외: 벨로르 "가품 판명 시 전액환불"은 확정 정책이므로 사용 가능.)
- 유치한 카피 금지(예: "기회 잘 잡으면 싸게 데려온다" 같은 문구 금지).
- 신뢰는 형용사가 아니라 **구체 정보**로 만든다: 절차 단계, 실제 숫자(연 매입 건수, 처리 기간), 사업자 정보, 실제 사례.
- 시계 매입 콘텐츠: 검색 시세와 실매입가 차이, 구성품/상태/연식/스탬핑에 따른 감가를 사실대로 설명. "높게 부르고 나중에 깎는" 화법 금지.

## 4. 금지사항 (하나라도 어기면 재작업)
- 시키지 않은 기능·섹션·파일 임의 추가 금지. 하지 말라는 것 절대 금지.
- 무료 스톡 느낌 템플릿 레이아웃(가운데 아이콘 3개 + 제목 + 회색 배경 반복) 금지.
- 첫 화면에 텍스트 5줄 이상 금지. 폴드 위에 CTA 없는 구성 금지.
- lorem ipsum·플레이스홀더 텍스트를 결과물에 남기지 말 것. 더미가 필요하면 실제 쓸 수 있는 한국어 문안을 넣는다.
- 반응형 미확인 제출 금지: 375px / 768px / 1280px 3개 폭에서 잘림·겹침·가로스크롤 없는지 확인 후 제출.

## 5. 제출 전 최종 자기검수 (답이 하나라도 '아니오'면 제출하지 말고 고쳐라)
1. 375px에서 스크롤 없이 정체성+이미지+CTA가 보이는가
2. 1차 CTA가 페이지에 1종류이고 즉시 눈에 들어오는가
3. 텍스트가 이미지보다 많아 보이는 섹션이 없는가
4. 같은 요소끼리 규격이 완전히 동일한가
5. 3개 폭에서 깨짐·잘림·가로스크롤이 없는가
6. 과장/보장 문구가 없는가
7. 각 섹션의 사업 목적을 한 단어로 답할 수 있는가
